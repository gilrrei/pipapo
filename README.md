# pipapo
**P**ython **I**mplemented **PA**rticles **PO**stprocessor

## Installation
1. Create a python environment, for example conda using:
   ```bash
   conda env create 
   ```
1. Install via:
   ```bash
   pip install .
   ```
   For a safe installation with fixed dependencies packages use:
   ```bash
   pip install .[safe]
   ```
   Develop install:
   ```bash
   pip install -e .[dev]
   ```
